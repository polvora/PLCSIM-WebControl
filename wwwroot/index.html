Network communication

Softbus works out of the box. Choose a TCP/IP mode + a host adapter only if you need real Ethernet communication with the PLCs.
⚠️ No authentication — intended for trusted, closed networks (the typical PLCSIM lab/VM setup). Anyone who can reach this page can power PLCs on/off. Don't place it on an untrusted network or expose it to the internet.
